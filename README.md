# Instagram Advanced Project

## Overview
Instagram Advanced is a feature-rich social media application that enhances user experience and introduces advanced functionalities.

## Features
- **User Profiles:** Create and edit personal profiles.
- **Image Sharing:** Upload and share photos.
- **Story Feature:** Post stories that disappear after 24 hours.
- **Messaging:** Direct messaging feature to communicate with friends.
- **Notifications:** Get updates about likes, comments, and messages.
- **Search:** Find users and posts easily.

## Tech Stack
- **Frontend:** React.js, Redux for state management
- **Backend:** Node.js with Express
- **Database:** MongoDB for data storage
- **Authentication:** JSON Web Tokens (JWT) for secure access
- **Hosting:** Deployed on Heroku

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/syedayyan03/instagram-advanced.git
   cd instagram-advanced
   ```
2. Install the necessary dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables. Create a `.env` file in the root directory and add the following:
   ```plaintext
   MONGODB_URI=your_mongodb_uri
   JWT_SECRET=your_jwt_secret
   ```
4. Run the application:
   ```bash
   npm start
   ```

## Usage Instructions
- Access the application through the provided URL after running it locally.
- Create a new account or log in to your existing profile to start using the features.
- Explore the application and share your feedback!
